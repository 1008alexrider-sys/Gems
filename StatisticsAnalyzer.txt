## Key Statistics They Analyze

| Metric | Significance |
|--------|--------------|
| **Execution Count** | How many times procedure was called |
| **Total Elapsed Time** | Cumulative execution time |
| **Total CPU Time** | CPU consumed |
| **Logical Reads** | Data pages accessed from memory |
| **Physical Reads** | Disk I/O operations |
| **Last Execution Time** | Most recent run duration |
| **Max Elapsed Time** | Worst-case execution time |
| **Avg Elapsed Time** | Average per execution |
| **Wait Time** | Time spent waiting for resources |
| **Plan Handle** | Query execution plan identifier |

## Step-by-Step Process: Comparing Stored Procedure Statistics

### **STEP 1: Capture Baseline Statistics (Time 1)**

```sql
-- Capture Statistics - Run this at first measurement point
SELECT 
    DB_NAME() AS DatabaseName,
    OBJECT_NAME(sp.object_id) AS ProcedureName,
    sp.object_id,
    sp.create_date,
    sp.modify_date,
    ps.execution_count,
    ps.total_elapsed_time / 1000000.0 AS total_elapsed_time_seconds,
    ps.total_cpu_time / 1000000.0 AS total_cpu_time_seconds,
    ps.total_logical_reads,
    ps.total_physical_reads,
    ps.total_worker_time / 1000000.0 AS total_worker_time_seconds,
    ps.last_execution_time,
    ps.min_elapsed_time / 1000.0 AS min_elapsed_time_ms,
    ps.max_elapsed_time / 1000.0 AS max_elapsed_time_ms,
    ps.last_elapsed_time / 1000.0 AS last_elapsed_time_ms,
    CAST(ps.total_elapsed_time AS FLOAT) / NULLIF(ps.execution_count, 0) / 1000.0 AS avg_elapsed_time_ms,
    CAST(ps.total_logical_reads AS FLOAT) / NULLIF(ps.execution_count, 0) AS avg_logical_reads
INTO Statistics_Baseline_Snapshot1
FROM sys.procedures sp
INNER JOIN sys.dm_exec_procedure_stats ps 
    ON sp.object_id = ps.object_id
WHERE ps.database_id = DB_ID()
ORDER BY ps.total_elapsed_time DESC;

-- Export this data to CSV for comparison
```

### **STEP 2: Implement Changes (Optimization)**

```sql
-- Example: Optimize a stored procedure
-- Original procedure (slow)
ALTER PROCEDURE usp_GetCustomerOrders
    @CustomerID INT
AS
BEGIN
    SELECT o.OrderID, o.OrderDate, o.TotalAmount
    FROM Orders o
    INNER JOIN Customers c ON o.CustomerID = c.CustomerID
    WHERE c.CustomerID = @CustomerID
    ORDER BY o.OrderDate DESC
END

-- Optimized version (with index hints, better filtering)
ALTER PROCEDURE usp_GetCustomerOrders
    @CustomerID INT
AS
BEGIN
    SELECT o.OrderID, o.OrderDate, o.TotalAmount
    FROM Orders o WITH (INDEX(idx_Orders_CustomerID))
    WHERE o.CustomerID = @CustomerID
    ORDER BY o.OrderDate DESC
END
```

### **STEP 4: Compare the Two Snapshots**

```sql
-- Compare Snapshot 1 vs Snapshot 2
SELECT 
    COALESCE(s1.ProcedureName, s2.ProcedureName) AS ProcedureName,
    s1.execution_count AS Executions_Before,
    s2.execution_count AS Executions_After,
    s2.execution_count - s1.execution_count AS Execution_Count_Change,
    
    ROUND(s1.avg_elapsed_time_ms, 2) AS Avg_Time_Before_ms,
    ROUND(s2.avg_elapsed_time_ms, 2) AS Avg_Time_After_ms,
    ROUND(s1.avg_elapsed_time_ms - s2.avg_elapsed_time_ms, 2) AS Avg_Time_Improvement_ms,
    
    CASE 
        WHEN s1.avg_elapsed_time_ms > 0 
        THEN ROUND(((s1.avg_elapsed_time_ms - s2.avg_elapsed_time_ms) / s1.avg_elapsed_time_ms * 100), 2)
        ELSE 0 
    END AS Performance_Improvement_Percent,
    
    s1.total_logical_reads AS Logical_Reads_Before,
    s2.total_logical_reads AS Logical_Reads_After,
    s2.total_logical_reads - s1.total_logical_reads AS Logical_Reads_Change,
    
    ROUND(CAST(s1.total_logical_reads AS FLOAT) / NULLIF(s1.execution_count, 0), 0) AS Avg_Logical_Reads_Before,
    ROUND(CAST(s2.total_logical_reads AS FLOAT) / NULLIF(s2.execution_count, 0), 0) AS Avg_Logical_Reads_After,
    
    s1.total_cpu_time_seconds AS CPU_Time_Before_Sec,
    s2.total_cpu_time_seconds AS CPU_Time_After_Sec,
    ROUND(s1.total_cpu_time_seconds - s2.total_cpu_time_seconds, 2) AS CPU_Time_Improvement_Sec,
    
    CASE 
        WHEN s2.avg_elapsed_time_ms < s1.avg_elapsed_time_ms THEN 'IMPROVED'
        WHEN s2.avg_elapsed_time_ms > s1.avg_elapsed_time_ms THEN 'DEGRADED'
        ELSE 'NO CHANGE'
    END AS Status
FROM Statistics_Baseline_Snapshot1 s1
FULL OUTER JOIN Statistics_Baseline_Snapshot2 s2
    ON s1.ProcedureName = s2.ProcedureName
ORDER BY ABS(COALESCE(s1.avg_elapsed_time_ms, 0) - COALESCE(s2.avg_elapsed_time_ms, 0)) DESC;
```

### **STEP 5: Export Comparison Data**

```sql
-- Export to staging table for PDF generation
SELECT 
    COALESCE(s1.ProcedureName, s2.ProcedureName) AS ProcedureName,
    s1.execution_count AS Executions_Before,
    s2.execution_count AS Executions_After,
    ROUND(s1.avg_elapsed_time_ms, 2) AS AvgTime_Before_ms,
    ROUND(s2.avg_elapsed_time_ms, 2) AS AvgTime_After_ms,
    CASE 
        WHEN s1.avg_elapsed_time_ms > 0 
        THEN ROUND(((s1.avg_elapsed_time_ms - s2.avg_elapsed_time_ms) / s1.avg_elapsed_time_ms * 100), 2)
        ELSE 0 
    END AS Improvement_Percent,
    ROUND(s1.total_cpu_time_seconds, 2) AS CPU_Time_Before_Sec,
    ROUND(s2.total_cpu_time_seconds, 2) AS CPU_Time_After_Sec,
    s1.max_elapsed_time_ms AS MaxTime_Before_ms,
    s2.max_elapsed_time_ms AS MaxTime_After_ms,
    s1.total_logical_reads AS LogicalReads_Before,
    s2.total_logical_reads AS LogicalReads_After
INTO StoredProc_Comparison_Report
FROM Statistics_Baseline_Snapshot1 s1
FULL OUTER JOIN Statistics_Baseline_Snapshot2 s2
    ON s1.ProcedureName = s2.ProcedureName
WHERE COALESCE(s1.ProcedureName, s2.ProcedureName) IS NOT NULL;
```
